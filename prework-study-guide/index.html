HTML

Key Phrases

  • Elements: Everything from a start tag to an end tag including it's content
  • Attributes: Provide additional information about an element (ie. src, width, height, alt)
  • Semantic Elements: An element that provides insight into what type of content is contained within (Acts as a container for content ie. form, table, header, footer)

HTML Tags

  • <head>: Contains infoformation about the webpage
  • <body>: Represents the visible content shown to the user
  • <h1>-<h6>: Various heading sizes from largest to smallest based on importance (note that h1 shall only be used once per page)
  • <p>: Text content shown to user
  • <img>: Adds an image

Git

Key Commands To Remember

  • git pull origin main: Pulls content from repository main branch to ensure code is up to date
  • git status: Displays current branch location and displays any modified files that need staged before commit
  • git checkout -b "branch-name": Creates a new branch and moves you into the new branch
  • git add -A: Stages all changes to files to be commited
  • git commit -m "message": Commits all staged changes along with a descriptive message
